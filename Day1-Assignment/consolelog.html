CONSOLE METHODS

Methods Explanation
1. console log() The console log() is used to print the Entered Value in inside the console log().
Ex : console log("Hello Javascript")
2. console error() The console error() is used to show the error message in the website. The default error message will be highlighted with red color. It is used to testing Code.
Ex : console error("an error occurred")
3. console warn() The console warn() is used to show the warning message in the website. The default warinng message will be highlighted with yellow color. It is used to testing Code.
Ex : console error("This is a warning Messege")
4. console table() The console table() is used to show a table of the Console. The input must be array which will be ahown as table.
Ex : console table({Name : "Kiran", Age : "23", City: "Chennai "})
5. console clear() The console clear() is used to clear the console.
Ex : console clear()
6. console time() & console timeEnd The console time() and timeEnd() is to know how much time is taken to function the console.
Ex : console time("Time Taken") & console timeEnd("Time Taken")
