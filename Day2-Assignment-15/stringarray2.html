String and Array

1. Array

The Array is used to store multiple values in single varriable.
Ex:
var fruits=["Mango", "Apple","Graps"];
console log(fruits);

2. String

The String is a primitive datatypes. The string can store charters as in one varriable.


Ex:
var Name="kiran"
console log(Name);

String has Sevaral Methods :

1. slice()

2. chartAt()

a). slice()

The slice() is a part of string methods. And it returns the new string. This methds has two parameters : start position, end position.


Ex:
var str="Kiran, Navi, Varsha"
var res=str slice(7,10);
console log(res);
