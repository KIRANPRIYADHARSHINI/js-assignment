Differnce between VAR AND LET AND CONST

Var Let Const
The var will be able to declaring the variable anywhere in the paricular block. And var is a global variable. var is "FUNCTION SCOPE".
Ex:
var name="kiran"
var age=24;
var canfly=false;
var languages= ['Tamil','English']; 		The Let will be a Local varriable, which will be the scobe of the block. Let will be used to work inside the particular scobe. And let is a "BLOCK SCOPED". The “block” defined by curly braces i.e. {}.
Ex:
let name="chinnu";
name="varsha";
console.log("Uisng Let: ",name); console.log(name); 		Constant is a keyword (const). The const cannot be re-assigned to a new value. And const is a "BLOCK SCOPED".
Ex:
const country = "India";
console.log(country);
